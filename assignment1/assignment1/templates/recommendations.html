Why do you decide to log user actions?

User actions give information about the behaviour of a user, his thinking, the prominent sections of a user interface and tons of other information that can be fed to intelligent systems that learn each user's special traits, likings, interests and help better serve the information that's available for the user, personalized to the specific user, dynamically, meaning that the user model keeps on evolving. Thus, logging user actions is an important input for an adaptive system with regards to implicit user information collection.
